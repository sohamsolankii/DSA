## Sorting Algorithms

1. **Bubble Sort** - A simple comparison-based sorting algorithm.
2. **Selection Sort** - A comparison-based algorithm that sorts by repeatedly finding the minimum element.
3. **Insertion Sort** - A comparison-based algorithm that builds the final sorted array one item at a time.
4. **Merge Sort** - A divide and conquer algorithm that divides the array into halves, sorts them, and merges them.
5. **Quick Sort** - A highly efficient sorting algorithm that uses partitioning.
6. **Heap Sort** - A comparison-based sorting technique based on a binary heap data structure.
7. **Radix Sort** - A non-comparative integer sorting algorithm.
8. **Bucket Sort** - A distribution sort that distributes elements into buckets.
9. **Counting Sort** - A non-comparative sorting algorithm that sorts by counting the number of occurrences of each element.

## Search Algorithms

1. **Linear Search** - A simple search algorithm that checks every element in the list.
2. **Binary Search** - A fast search algorithm that works on sorted arrays by repeatedly dividing the search interval in half.
3. **Depth-First Search (DFS)** - An algorithm for traversing or searching tree or graph data structures.
4. **Breadth-First Search (BFS)** - An algorithm for searching a tree or graph data structures level by level.
5. **Jump Search** - An algorithm for searching sorted arrays by jumping ahead by fixed steps.
6. **Exponential Search** - An algorithm that works on sorted arrays and uses a combination of binary search and exponential jumps.
7. **Interpolation Search** - An improved variant of binary search best suited for uniformly distributed data.

## Graph Algorithms

1. **Dijkstra’s Algorithm** - An algorithm for finding the shortest paths between nodes in a graph.
2. **Bellman-Ford Algorithm** - An algorithm for finding the shortest paths in a weighted graph.
3. **Floyd-Warshall Algorithm** - An algorithm for finding shortest paths in a weighted graph with positive or negative edge weights.
4. **A\* Search Algorithm** - An algorithm that finds the shortest path from a start node to a goal node.
5. **Kruskal’s Algorithm** - A minimum spanning tree algorithm that finds an edge of the least possible weight.
6. **Prim’s Algorithm** - A minimum spanning tree algorithm that starts with a single vertex.
7. **Topological Sort** - An algorithm for ordering vertices in a directed acyclic graph.
8. **Tarjan's Algorithm** - An algorithm for finding strongly connected components in a graph.
9. **Kosaraju's Algorithm** - Another algorithm for finding strongly connected components in a directed graph.

## Dynamic Programming Algorithms

1. **Fibonacci Sequence** - An algorithm for generating the Fibonacci sequence.
2. **Knapsack Problem** - An algorithm for solving the knapsack problem.
3. **Longest Common Subsequence** - An algorithm for finding the longest subsequence common to all sequences.
4. **Longest Increasing Subsequence** - An algorithm for finding the longest increasing subsequence.
5. **Matrix Chain Multiplication** - An algorithm for finding the optimal way to multiply matrices.
6. **Edit Distance** - An algorithm for finding the minimum number of edits required to convert one string into another.
7. **Coin Change Problem** - An algorithm for finding the minimum number of coins needed to make a certain amount of change.

## Divide and Conquer Algorithms

1. **Merge Sort** - An efficient sorting algorithm that divides the array into halves.
2. **Quick Sort** - A sorting algorithm that uses partitioning.
3. **Strassen’s Matrix Multiplication** - An algorithm for matrix multiplication.
4. **Binary Search** - A fast search algorithm for sorted arrays.
5. **Closest Pair of Points** - An algorithm for finding the closest pair of points in a plane.

## Greedy Algorithms

1. **Huffman Coding** - An algorithm for lossless data compression.
2. **Kruskal’s Algorithm** - An algorithm for finding the minimum spanning tree.
3. **Prim’s Algorithm** - An algorithm for finding the minimum spanning tree.
4. **Activity Selection Problem** - An algorithm for selecting the maximum number of activities that don't overlap.
5. **Fractional Knapsack Problem** - An algorithm for maximizing the total value in the knapsack.

## Backtracking Algorithms

1. **N-Queens Problem** - An algorithm for solving the N-Queens problem.
2. **Sudoku Solver** - An algorithm for solving Sudoku puzzles.
3. **Knight’s Tour Problem** - An algorithm for finding the Knight's tour on a chessboard.
4. **Hamiltonian Path Problem** - An algorithm for finding a Hamiltonian path in a graph.
5. **Subset Sum Problem** - An algorithm for finding a subset of numbers that sum to a given value.

## String Algorithms

1. **KMP Algorithm** - An algorithm for substring search.
2. **Rabin-Karp Algorithm** - An algorithm for searching multiple patterns.
3. **Boyer-Moore Algorithm** - An algorithm for efficient string searching.
4. **Z Algorithm** - An algorithm for pattern matching.
5. **Aho-Corasick Algorithm** - An algorithm for matching a set of patterns in a string.

2. **Topological Sort** - An algorithm for ordering vertices in a directed acyclic graph.
3. **K-Means Clustering** - An algorithm for partitioning n observations into k clusters.
4. **PageRank Algorithm** - An algorithm used by Google Search to rank web pages.
5. **Fast Fourier Transform (FFT)** - An algorithm for computing the discrete Fourier transform.
6. **RSA Algorithm** - An algorithm for public-key cryptography.
